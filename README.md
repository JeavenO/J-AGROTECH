company focused on smart farming and sustainable agriculture.

- Modern responsive design
- Glass-effect navigation bar
- Scroll animations
- Animated statistics
- Product showcase
- Contact form
- Multi-page website

## Technologies Used

- HTML5
- CSS3
- JavaScript
